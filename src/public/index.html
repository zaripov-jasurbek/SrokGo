SrokGo API Playground

Быстрый UI для проверки логики API. Выберите эндпоинт ниже или заполните форму вручную.

Запрос

Например: http://localhost:3000
Если выбран файл, запрос отправится как multipart/form-data.

Ответ

Нажмите «Отправить», чтобы увидеть ответ.

Эндпоинты API

Нажмите на эндпоинт, чтобы заполнить форму.
